# Интеграции — Discord

## Почему Discord

CrossCrypt — игра, которая полностью разворачивается внутри Discord: игроки уже находятся там, общаются голосом и взаимодействуют через команды бота. Встроенный голосовой канал снимает необходимость реализовывать собственный voice-чат. Discord OAuth2 обеспечивает готовую систему авторизации без хранения паролей.

## Что используем

- **OAuth2 Authorization Code Flow** — авторизация пользователя через Discord-аккаунт; получение `access_token` и идентификатора пользователя (`discord_id`)
- **GET /users/@me** — получение профиля пользователя (username, avatar) после авторизации
- **Discord Gateway (Bot)** — доставка сообщений, кнопок и embed-контента игрокам; обработка slash-команд и взаимодействий
- **Discord Voice** — голосовой канал для коммуникации команды во время сессии; CrossCrypt не управляет голосом напрямую, игроки подключаются к каналу самостоятельно
- **Webhook (опционально)** — отправка результата сессии (победа/поражение) в Discord-канал команды

## Решение об интеграции

Игра реализована как Discord-бот: все действия выполняются через slash-команды и интерактивные кнопки внутри Discord. Авторизация через Discord исключает отдельную регистрацию и снижает порог входа, сохраняя пользователя в привычной экосистеме.

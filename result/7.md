# Бизнес-правила CrossCrypt

1. **Минимальный состав команды** — игровая сессия стартует только при наличии ровно трёх подключённых игроков; с двумя или менее игра не начинается

2. **Уникальность ролей** — каждая из трёх ролей (Хакер, Аналитик, Криптограф) назначается ровно одному игроку; дублирование роли в рамках одной сессии недопустимо

3. **Закрытость сессии после старта** — после начала игры подключение новых игроков заблокировано; состав команды фиксируется в момент запуска

4. **Информационная изоляция ролей** — каждый игрок видит только те данные, которые соответствуют его роли; доступ к информации других ролей технически исключён

5. **Система штрафов** — неверный ввод символа в кроссворд фиксируется как ошибка и сокращает оставшееся время раунда; победа возможна только при полностью заполненном кроссворде до истечения таймера

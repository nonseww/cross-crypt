# Общая логика системы

Система состоит из четырёх слоёв:

- **Discord Client** — игровой интерфейс внутри Discord: команды, кнопки, embed-сообщения; каждый из трёх игроков видит только свой контент, изолированный по роли
- **Backend** — единый сервер, обслуживающий REST API (CRUD, управление сессией) и WebSocket (real-time синхронизация поля, таймера, штрафов)
- **PostgreSQL** — постоянное хранилище: комнаты, сессии, кроссворды, ключи шифрования, штрафы
- **Discord API** — внешний сервис: OAuth2-авторизация пользователей, доставка сообщений и взаимодействий через Gateway

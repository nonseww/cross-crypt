# API

## CrossCrypt API — ключевые методы

---

**POST /api/rooms**
Создание игровой комнаты

Параметры запроса:
```json
{ "userId": "uuid" }
```

Пример ответа:
```json
{
  "roomId": "uuid",
  "roomCode": "A4F9X1",
  "status": "waiting"
}
```

---

**POST /api/rooms/{roomCode}/join**
Подключение игрока по коду комнаты

Параметры запроса:
```json
{ "userId": "uuid" }
```

Пример ответа:
```json
{
  "playerId": "uuid",
  "roomId": "uuid",
  "role": null,
}
```

---

**POST /api/sessions/{sessionId}/start**
Запуск игровой сессии (после подключения 3 игроков)

Параметры запроса:
```json
{ "initiatorId": "uuid" }
```

Пример ответа:
```json
{
  "sessionId": "uuid",
  "status": "in_progress",
  "players": [
    {
      "playerId": "uuid",
      "role": "hacker"
    },
    {
      "playerId": "uuid",
      "role": "analyst"
    },
    {
      "playerId": "uuid",
      "role": "cryptographer"
    }
  ],
  "timerSeconds": 600
}
```

---

**PATCH /api/sessions/{sessionId}/crossword/cells/{cellId}**
Ввод/редактирование символа в ячейке кроссворда (доступно роли Hacker)

Параметры запроса:
```json
{
  "userId": "uuid",
  "value": "Ж"
}
```

Пример ответа:
```json
{
  "cellId": "uuid",
  "value": "Ж",
  "progress": 0.42
}
```

---

**GET /api/sessions/{sessionId}/cipher-key**
Получение текущей таблицы шифрования (доступно роли Cryptographer)

Параметры запроса:
```
playerId: uuid (query)
```

Пример ответа:
```json
{
  "version": 3,
  "mapping": {
    "А": "#",
    "Б": "@",
    "В": "%"
  },
  "updatedAt": "2026-06-15T10:32:00Z"
}
```
